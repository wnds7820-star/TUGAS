## 🔧 Customization

### Change Pass Threshold
Edit in `wnds_stats_pro.py`:
```python
self.pass_threshold = 75  # Change to your value
```

### Modify Colors
Edit in `create_styles()`:
```python
onyx_black = "#0a0a0a"
phosphor_green = "#00ff41"
```

### Update Login
Edit in `verify_login()`:
```python
if username == "admin" and password == "secret":
```

## 🚨 Troubleshooting

**Won't start?**
```bash
# Check Python version
.venv\Scripts\python.exe --version
```

**Charts not showing?**
```bash
pip install matplotlib --upgrade
```

**PDF export fails?**
```bash
pip install reportlab
```

**Slow performance?**
- Reduce number of records
- Clear activity log occasionally
- Restart application
